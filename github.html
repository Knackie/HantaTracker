Déploiement GitHub Pages

Ce site est conçu pour être publié directement depuis un dépôt GitHub, sans backend et sans build complexe.

Déployer

Crée un dépôt GitHub, envoie ces fichiers à la racine, puis active GitHub Pages dans les paramètres du dépôt. La source doit être la branche main et le dossier /root.

Commandes

git init
git add .
git commit -m "Initial HantaTracker"
git branch -M main
git remote add origin https://github.com/TON_USER/hantatracker.git
git push -u origin main

URL finale

Après activation de GitHub Pages, le site sera disponible à l’adresse https://TON_USER.github.io/hantatracker/.
