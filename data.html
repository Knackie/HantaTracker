Données et définitions

Le tableau de bord distingue les cas documentés, les cas confirmés et les cas probables. La carte n’invente pas de cas par pays.

Source principale

Organisation mondiale de la Santé

Le script lit la page OMS DON600 configurée dans scripts/update-data.js. Si la page change, le fichier JSON est mis à jour automatiquement.

Voir la source OMS →

Définitions

Cas documentés, pas “associés”

Cas documentés signifie : cas publiés dans la source officielle utilisée. Ce n’est pas une estimation globale.

Cas probables reprend la formulation OMS lorsqu’un cas est documenté mais non confirmé en laboratoire.

France

Santé publique France

Les données françaises doivent être traitées séparément via Santé publique France / Odissé, car elles relèvent d’une surveillance territoriale différente.
