Méthodologie

Données et sources

Cette page explique d’où viennent les chiffres, comment ils sont stockés et comment les mettre à jour.

Source principale

Organisation mondiale de la Santé

Le jeu de données initial est basé sur un bulletin Disease Outbreak News de l’OMS. Le fichier local data/hantavirus.json contient les valeurs utilisées par le tableau de bord.

Voir les bulletins OMS →

Format

JSON public

Les données sont volontairement simples : date de mise à jour, source, cas totaux, cas confirmés, cas suspects, décès et série temporelle. 

{
  "totalCases": 8,
  "confirmedCases": 6,
  "suspectedCases": 2,
  "deaths": 3
}

Important

Lecture prudente des chiffres

Les chiffres affichés concernent uniquement les signalements documentés dans la source utilisée. Ils ne représentent pas une estimation exhaustive de tous les cas possibles dans le monde.
