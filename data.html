Méthodologie

Données et sources

Les chiffres sont stockés dans un fichier JSON public et mis à jour par GitHub Actions à partir de la page OMS configurée.

Source principale

Organisation mondiale de la Santé

Le script lit la page Disease Outbreak News configurée dans scripts/update-data.js.

Voir la source OMS →

Format

JSON public

Le fichier contient les indicateurs, la série temporelle et les coordonnées utilisées pour la carte.

{
  "totalCases": 8,
  "confirmedCases": 6,
  "deaths": 3,
  "locations": []
}

Auto-update

Mise à jour quotidienne

Le workflow .github/workflows/update-data.yml peut être lancé manuellement ou automatiquement chaque matin.
